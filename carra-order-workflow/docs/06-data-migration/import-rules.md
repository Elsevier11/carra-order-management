# Import Rules

- **MIG-001** — Ogni riga Excel diventa un record ordine unico.
- **MIG-002** — Il foglio di provenienza iniziale determina lo stato workflow iniziale.
- **MIG-003** — Le note esistenti vengono importate come nota storica iniziale.
- **MIG-004** — I valori testuali negli accessori vengono mappati in task quando riconosciuti da dizionario.
- **MIG-005** — I campi non strutturati non devono bloccare l'import, ma essere marcati per revisione.
- **MIG-006** — Gli ordini nei fogli storici devono essere importabili con stato chiuso o archiviato.
