# Kanban and List Views

## Vista board

Adatta a coordinamento operativo giornaliero, avanzamento per stato e colpo d'occhio.

## Vista lista

Adatta a ricerca dettagliata, controllo campi, confronto date, export e attività amministrative.

## Regola progettuale

Le due viste devono insistere sullo stesso dataset ordini, senza duplicazione logica.
