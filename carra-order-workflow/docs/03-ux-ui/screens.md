# Screens

## UI-001 Dashboard workflow

Scopo:
- visualizzare ordini per stato;
- filtrare rapidamente gli ordini attivi;
- individuare urgenze e blocchi.

Componenti:
- board kanban per stato;
- filtri rapidi;
- badge urgenza/blocco;
- contatori per colonna.

## UI-002 Scheda ordine

Scopo:
- gestire tutti i dati di un ordine in un'unica vista.

Sezioni:
- dati principali ordine;
- stato workflow e stato disegno;
- assegnazioni;
- task accessori;
- note cronologiche;
- storico attività.

## UI-003 Vista consegne

Scopo:
- pianificare consegne e A.M.P.

Contenuti:
- ordini pronti e avvisati;
- data/ora consegna;
- trasportatore;
- camion;
- stato logistico.

## UI-004 Vista tabellare ordini

Scopo:
- supportare ricerca, filtro massivo, export e controllo dettagliato.
