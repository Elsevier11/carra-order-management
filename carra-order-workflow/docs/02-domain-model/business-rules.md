# Business Rules

- **WF-003** — Non è consentito iniziare la lavorazione senza conferma del disegno.
- **WF-004** — Gli accessori critici possono generare task bloccanti.
- **WF-005** — Gli ordini sospesi devono riportare motivo, data e nota di follow-up.
- **WF-006** — Il passaggio a pronto e avvisato richiede dati minimi di consegna.
- **WF-007** — Tutte le note operative devono essere storicizzate.
- **WF-008** — Le date chiave devono essere tracciate separatamente per audit operativo.
