# Glossary

- **Ordine** — Commessa o ordine cliente gestito nel workflow operativo.
- **Disegno tecnico** — Disegno esecutivo da creare, inviare e far confermare al cliente.
- **A.M.P.** — Avviso Merce Pronta.
- **Accessori critici** — Elementi che generano vincoli o lavorazioni aggiuntive, ad esempio calcoli o ferri.
- **Sospeso** — Ordine bloccato o fermo per causa esterna o interna.
- **Bancali** — Colli/accessori da preparare oltre alle vasche.
